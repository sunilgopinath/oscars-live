## Simple App Setup

```js
git clone https://github.com/sunilgopinath/oscars-live.git
npm install
node app.js
```
Now browse to http://localhost:7076

### Technologies used

#### express.io (realtime-web framework for node.js)
```coffeescript
express.io = express + socket.io
```
